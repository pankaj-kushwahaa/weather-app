How to access Weather API

{{url}}weather/api?lat={Latitude}&lon={Longitude}&detailing-type={minutely}
detailing-type
minutely
hourly
daily
Minutely forecast for an Hour
Hourly forecast for 48 hours
Daily forecast for 8 days

Example:
Minutely: {{url}}weather/api?lat=30.732962&lon=76.779175&detailing-type=minutely

Hourly: {{url}}weather/api?lat=30.732962&lon=76.779175&detailing-type=hourly

Daily: {{url}}weather/api?lat=30.732962&lon=76.779175&detailing-type=daily
